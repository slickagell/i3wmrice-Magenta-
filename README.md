## Dependencies
* Base i3           : `i3wm i3-gaps i3status i3lock i3lock-color`
* Terminal          : `rxvt-unicode`
* Applet            : `nm-applet volumeicon`
* Solid BG          : `xsetroot`
* App launcher      : `rofi`
* Screenshot        : `scrot`
* Font              : Monospace, Nerd Font Mono
